# Kursovoy_project


1. Табличные данные (кейс №9 — хоккейная статистика)
Название: NHL Player Database

Ссылка: https://www.kaggle.com/datasets/flynn28/nhl-player-database


2. Временные ряды (кейс №33 — распознавание событий)
Название: Human Activity Segmentation Challenge Dataset

Ссылка: https://github.com/patrickzib/human_activity_segmentation_challenge


3. Изображения (кейс №10 — распознавание оружия)
Название: Weapon Detection Dataset

Ссылка: https://www.kaggle.com/datasets/snehilsanyal/weapon-detection-test


4. Текстовые данные (кейс №49 — суммаризация документов)
Название: CNN / Daily Mail Dataset for Text Summarization

Ссылка: https://huggingface.co/datasets/cnn_dailymail
Ссылка 2: https://www.kaggle.com/datasets/gowrishankarp/newspaper-text-summarization-cnn-dailymail


5. Аудиоданные (кейс №93 — голосовой помощник)
Название: Multilingual Speech Command Recognition Dataset (русская часть)

Ссылка: https://github.com/IS2AI/Multilingual-Speech-Command-Recognition
